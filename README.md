OrderManagement repository

Order Management API (ASP.NET Core, .NET 9)

	Designed and implemented a RESTful backend system for managing clients, suppliers, goods, and orders.

	Key Features:
		Designed relational database model with multiple entities and relationships
		Implemented CRUD operations for all entities
		Applied DTO pattern to separate API contracts from database models
		Built clean architecture using Repository and Service layers
		Integrated Entity Framework Core with SQL Server
		Implemented automatic database seeding
		Configured OpenAPI + Swagger UI for API testing

	Tech: C#, ASP.NET Core, EF Core, SQL Server, Swagger
